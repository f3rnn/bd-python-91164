# Aula de CRUD - Banco de Dados.

## Objetivo:
Criar conexão com banco de dados e realizar operações básicas (CRUD).

### Teconologias:
- ORM: SQLAlchemy
- Banco de Dados: SQLite

### Detalhes:
Criar conexão com banco de dados SQLite, usando o SQLAlchemy como ferramenta ORM para interação com banco de dados.
